# Flutter Task Manager App

## 📌 Project Description
A simple Task Manager application built using Flutter.  
It allows users to add, delete, and search tasks with persistent storage using SharedPreferences.

## 🚀 Features
- Add tasks
- Delete tasks
- Search tasks in real-time
- Category selection (Work, Personal, Study)
- Persistent storage (tasks remain after restart)

## 🛠 Tech Stack
- Flutter
- Dart
- SharedPreferences

## 📱 Platform
- Android (Tested via FlutLab)
